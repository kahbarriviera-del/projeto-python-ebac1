# Projeto Python - EBAC

## Descrição

Projeto desenvolvido durante o curso de Análise de Dados da EBAC utilizando Python e GitHub.

## Arquivos do Projeto

- app.py
- executar.sh

## Como executar

### Linux ou Mac

Conceda permissão ao arquivo:

```bash
chmod +x executar.sh
```

Execute:

```bash
./executar.sh
```

### Windows

Execute diretamente o arquivo Python:

```bash
python app.py
```

## Explicação do Código

O programa solicita que o usuário digite seu nome e exibe uma mensagem personalizada de boas-vindas.

Fluxo do programa:

1. Recebe o nome digitado pelo usuário.
2. Armazena o valor na variável `nome`.
3. Exibe uma mensagem utilizando f-string.

Exemplo:

Entrada:

Maria

Saída:

Olá, Maria! Bem-vindo ao projeto EBAC.

## Tecnologias Utilizadas

- Python
- Git
- GitHub
- Shell Script (.sh)
